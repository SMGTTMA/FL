# 项目技术栈文档

## 核心技术

- React 18.2.0 - 前端核心框架
- TypeScript - 开发语言
- Vite - 构建工具

## UI 框架和样式

- Ant Design (antd) 5.x - UI 组件库
- Tailwind CSS - 原子化 CSS 框架
- styled-components - CSS-in-JS 解决方案
- @vanilla-extract/css - CSS 方案
- Framer Motion - 动画库
- classnames & clsx - 类名处理工具

## 状态管理和数据获取

- Zustand - 状态管理
- @tanstack/react-query - 数据请求和缓存
- Axios - HTTP 客户端

## 路由

- React Router - 路由管理

## 功能组件和工具库

- @dnd-kit/\* - 拖拽功能
- @fullcalendar/\* - 日历组件
- react-quill - 富文本编辑器
- react-markdown - Markdown 渲染
- react-organizational-chart - 组织架构图
- highlight.js - 代码高亮
- dayjs - 日期处理
- numeral - 数字格式化
- ramda - 函数式编程工具库

## 国际化

- i18next - 国际化解决方案
- react-i18next - React 国际化集成
- i18next-browser-languagedetector - 浏览器语言检测

## 开发工具和工程化

- @biomejs/biome - 代码格式化和检查
- commitlint - Git 提交信息规范
- lefthook - Git hooks 管理
- msw - 接口模拟
- postcss - CSS 处理器
- rollup-plugin-visualizer - 打包分析

## 性能和监控

- @vercel/analytics - 性能分析
- react-error-boundary - 错误边界处理
- nprogress - 加载进度条

## 字体和图标

- @fontsource-variable/\* - 可变字体
- @ant-design/icons - Ant Design 图标
- @iconify/react - Iconify 图标
- react-icons - 图标库

## 其他工具

- simplebar-react - 自定义滚动条
- screenfull - 全屏功能
- sonner - 通知提示
- react-helmet-async - 文档头部管理

## 开发环境要求

- Node.js: 20.x
- 包管理器: pnpm 9.1.0

## 项目特点

1. 使用现代化的技术栈，包含了完整的 TypeScript 支持
2. 完善的 UI 组件体系，结合了多个成熟的 UI 库
3. 强大的状态管理和数据获取方案
4. 完整的国际化支持
5. 丰富的功能组件支持（图表、富文本、日历等）
6. 完善的工程化配置，包含代码规范、提交规范等
7. 性能优化和监控方案
